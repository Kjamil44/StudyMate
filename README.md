# StudyMate
Тимски проект по Управување на ИКТ проекти - Тим 5

# Setup project
## 1. Clone repository
```
git clone https://github.com/Kjamil44/StudyMate.git
```
## 2. Setup react app
```
cd study-mate-ui
npm install
npm run start
```
## 3. Setup dotnet app
```
cd StudyMate.API
dotnet watch run
```
